Hi, I'm Peter Hurford. I work at Rise Interactive as a Lead Data Scientist in Chicago, using machine learning to help companies match customers with the products they are most likely to buy. Previously, I worked at Avant, using machine learning to reduce default rates and increase returns on marketing.

I'm an avid open source developer. Recently, I made Vowpal Platypus, a general use, lightweight Python wrapper built on Vowpal Wabbit, that enables quick, accurate, versitile, lightweight, multi-core, out-of-core machine learning in Python. I've also made Surveytools2, which makes it easier to analyze survey data; Checkr, which brings type safety and automatic verification to R; and Batchman, which lets you write batch functions in R. I also work with Vim and Bash: Vimflow brings Vim a quick navigation system and Git-it-on makes it easy to navigate GitHub from the command line.

By night, I'm on the board of Charity Science Health, a non-profit that uses SMS reminders to boost immunization rates in India. I'm also on the board of Animal Charity Evaluators, a non-profit that researches the best way to help nonhuman animals (I do research). I'm also the co-founder and advisor to .impact, a non-profit building an international community of people who want to make the world a better place (a topic I've given a TEDx talk about).

I blog across the internet and also occasionally tweet. I also read and watch the occasional TV. I do not play the organ.
